# Tres en Raya - React

Aplicación web del juego tres en raya desarrollada con React.

## 📌 Características
- Lógica del juego implementada en JavaScript
- Interfaz interactiva
- Control de turnos entre jugadores
- Diseño simple y funcional

## 🛠️ Tecnologías utilizadas
- React
- JavaScript
- Vite

## 🚀 Estado del proyecto
Proyecto desarrollado como parte de formación en desarrollo web.
